# Clouds Theme
I like clouds so I made a Hugo theme around it

# Getting Started
First install Hugo Extended by following these [instructions](https://gohugo.io/installation/) 

Then install and start up Tailwind
```pwsh
npm install

# Continuosly compiles Tailwind into CSS on file changes - for development 
npm run watch

# Compiles Tailwind into CSS once - for production
npm run build
```

Start up Hugo by running
```pwsh
# -D includes draft content to preview during development
hugo server -D
```
